# 🌐 Portfolio - Gian Franco Gatto

Este es mi portfolio personal desarrollado con **ReactJS** + **Vite**, con un diseño visual fuerte y profesional utilizando **Tailwind CSS** orientado a mostrar mis habilidades, experiencia y proyectos en desarrollo web.

## 🚀 Tecnologías utilizadas

- ⚛️ ReactJS
- ⚡ Vite
- 🎨 Tailwind CSS
- 🔧 Git & GitHub

## 🧱 Estructura del portfolio

- **Inicio**: presentación general.
- **Sobre mí**: formación, perfil profesional y objetivos.
- **Proyectos**: sección con trabajos académicos y freelance.
- **Experiencia**: historial laboral y tareas destacadas.
- **Contacto**: formulario e información para conectar.

## 📷 Diseño

Incluye un **banner personalizado estilo "developer"**, en formato de código, optimizado para LinkedIn. El sitio es totalmente responsive y con enfoque visual moderno.

## 🗂 Organización

- Proyecto creado con `npm create vite@latest`.
- Versionado con Git y sincronizado en GitHub.
- Preparado para ser desplegado en plataformas como **Vercel** o **Netlify**.
